# VeChain Academy NFT Minter

## Quick Start

### 1. Install Dependencies
```bash
yarn install
```

### 2. Start VeChain Solo Network
```bash
cd apps/contracts
yarn solo-up
```

### 3. Compile Smart Contract
```bash
yarn compile
```

### 4. Deploy to Solo Network
```bash
yarn deploy-solo
```

### 5. Run the mint script
```
bash
yarn mint-nft
```
